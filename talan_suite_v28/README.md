# talan_suite (parser-only)

UI полностью удалён — будем делать с 0.

## Что запускать

### 1) Запуск всех парсеров через запускатор

1. Открой папку `launcher`.
2. Запусти `run_launcher.bat`.

Запускатор:
- создаёт `launcher/data/published_snapshots/<RUN_ID>/` и `launcher/data/quarantine/<RUN_ID>/`
- запускает парсеры по конфигу
- «зелёные» снапшоты кладёт в `published_snapshots/<RUN_ID>/` без подпапок
- «подозрительные/красные» — в `quarantine/<RUN_ID>/`
- пишет общий лог в `launcher/logs/`.

### 2) Запуск конкретного парсера отдельно

Каждый парсер лежит в `parser/<developer>/` и содержит свой батник:
- `run_only_*.bat` — запускает **только парсер** и сохраняет снапшот в **его локальную** папку `parser/<developer>/output_*/...`.
- `run_*_auto.bat` (если есть) — то же самое, но с авто-ретраями и/или авто-получением токена (если требуется).

## Где смотреть результаты

- **Опубликованные снапшоты (готово для последующей нормализации/UI):**
  - `launcher/data/published_snapshots/<RUN_ID>/`
- **Карантин (нужна проверка человеком):**
  - `launcher/data/quarantine/<RUN_ID>/`
- **Логи:**
  - общий лог: `launcher/logs/`
  - логи парсеров: внутри `parser/<developer>/output_*/logs/`.
